#### JavaScript 3D library ####

The aim of the project is to create an easy to use, lightweight, 3D library. The library provides &lt;canvas&gt;, &lt;svg&gt;, CSS3D and WebGL renderers.

## Download and Link

Before you execute the steps below, please make sure you have the following installed and configured:

- Git
- Node
- npm

 __Steps:__
```
git clone https://github.com/ykamo001/three.js.git
cd ./three.js
npm install
npm run build-uglify
npm link
```
